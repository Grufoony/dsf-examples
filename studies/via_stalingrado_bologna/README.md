# Study of Via Stalingrado, Bologna, Italy
To run the simulation:
```shell
uv run via_stalingrado_bologna.py
```

This will create a new `output` folder with all the created files.

## Motivation
Via Staligrado in Bologna is a long straight road with a lot of traffic lights on it.
This is the perfect case to study the effect of traffic lights over the vehicle flow.

## Data
Input flow data are an interpolated version of the coil data available [here](https://opendata.comune.bologna.it/pages/home/).

Traffic lights phasings were measured manually in July 2022.
